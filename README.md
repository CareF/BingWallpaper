# BingWallpaper

将当天的必应美图设置为桌面背景（支持图形化和命令行设置）

## 使用

```bash 
## (需提前安装 Python 的 argparse requests)
python3 BingWallpaper.py -d [desktop-environment-name] //暂时支持 xfce,cinnamon,deepin欢迎 PR 添加其他桌面环境
python3 BingWallpaper.py -c [set-desktop-wallpaper-command] // 尚未支持
```

建议 Ubuntu/Linuxmint 用户下载 release 里的安装包，安装后使用图形化界面一键设置
